AskMeBazaar-Sales-Analysis-Dashboard-Power-BI-

Developed an interactive Sales Analysis Dashboard in Power BI using AskMeBazaar sales data. The project involved data cleaning, data modeling, KPI creation, and dashboard development to analyze sales performance, revenue, discounts, orders, customer purchasing behavior, payment modes, product performance, and geographical sales distribution. The dashboard enables business users to monitor key metrics and identify high-performing products, categories, cities, and states for data-driven decision-making.
